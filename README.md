API backend desenvolvida em Python para gerenciamento de clientes e simulação de integração com Pipefy via GraphQL.
A aplicação realiza cadastro de clientes, validação de dados, persistência em banco local, processamento de webhooks, definição automática de prioridade com base no patrimônio e controle de idempotência para evitar eventos duplicados.

O projeto também inclui testes automatizados, estrutura organizada em camadas e simulação das mutations GraphQL utilizadas pelo Pipefy para criação e atualização de cards.
